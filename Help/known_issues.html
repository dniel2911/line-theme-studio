Known Issues/Problems

As this program still on beta stage, still had many common problem:
  • For theme specification reason, only *.png inside picture resource
  • Image on property editor and resource editor thumbnail not updated until you refresh or select to another item then select again.
  • External editor not yet changeable. It will enable in version 1.2.x of release
  • You must install Google USB driver (can from original Android SDK or 15 seconds ADB installer), and your device USB driver to use send to ADB feature.
