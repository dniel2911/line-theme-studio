Quick Introduction


Line Theme Studio is program that assist you to make or edit infamous Line Messenger. It's intended easy to use and doesn't need any knowledge about internal theme file.

Some cool features are:

  • Edit or recreate from current theme
  • Easy to edit theme properties like colors, pictures and strings
  • Replace any current theme resources
  • Send  theme to your device or emulator via ADB*
  • And many more...

* Needs ADB driver and your device USB driver installed to use this feature.
